# Planner

Plan without panic!

![Demo](docs/demo.png)

## Features

* Checklist
* Habit management
* Schedule management

## Getting Started

```bash
git clone git@github.com:srg-bnd/planner.git & cd planner
yarn install
bundle install
rails db:migrate
rails server
```
